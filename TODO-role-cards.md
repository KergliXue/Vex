## 角色目录格式草案

```text
roles/<role-id>/
├── config.json
├── soul.md
└── avatar.png
```

```json
{
  "name": "Vex Classic",
  "description": "默认内置角色，继承原有 soul.md 灵魂设定。",
  "image": "avatar.png"
}
```
